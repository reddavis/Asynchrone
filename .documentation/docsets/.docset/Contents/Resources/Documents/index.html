Requirements

  • iOS 15.0+

Installation

Swift Package Manager

In Xcode:

  1. Click Project.
  2. Click Package Dependencies.
  3. Click +.
  4. Enter package URL: https://github.com/reddavis/Redux.
  5. Add Asynchone to your app target.
